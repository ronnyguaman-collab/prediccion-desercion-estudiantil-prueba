# Predicción de Deserción Estudiantil

Este proyecto desarrolla un sistema de predicción de deserción estudiantil utilizando técnicas de minería de datos y machine learning.

## Tecnologías
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## Modelo
- Regresión Logística
- Variables académicas: promedio, asistencia e índice de riesgo académico

## Ejecución
1. Instalar dependencias:
   pip install -r requirements.txt
2. Ejecutar la aplicación:
   streamlit run app.py
